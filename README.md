# 🎵 LyricTypeTest

> Typing test seru dengan sentuhan musik — ketik lirik lagu favoritmu secepat mungkin!

**LyricTypeTest** adalah aplikasi web sederhana yang terinspirasi dari *10FastFingers*, namun dengan twist unik:  
🔥 kata-kata yang diketik berasal dari **lirik lagu populer**.

---

## 🚀 Features

- 🎶 Typing berbasis lirik lagu
- ⏱️ Timer real-time
- ⚡ WPM (Words Per Minute) calculation
- 🎯 Accuracy tracking
- 🎧 Feel like you're typing along with your favorite song
